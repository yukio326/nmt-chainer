# nmt-chainer 

WAT2017におけるTMU（首都大小町研）のシステムです。
モデルの詳細は
[Tokyo Metropolitan University Neural Machine Translation System for WAT2017](http://aclweb.org/anthology/W17-5716.pdf)
を参照。
デモは
[こちら](http://cl.sd.tmu.ac.jp/~matsumura/translator)
。

## 1. 環境設定
- Python 3.5.1
- chainer (ver 1.24.0)
- h5py (ver 2.7.0)
- gensim (ver 2.2.0)
